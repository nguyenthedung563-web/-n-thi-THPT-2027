📋 Thu thập & chuẩn hóa dữ liệu đề thi

Nhập từ ảnh / chụp trực tiếp / link / file (PDF, Word, Excel...) → chuẩn hóa thành bảng → đẩy lên Google Sheets

⚠️ Bạn đang mở file này trực tiếp (file://...) — một số trình duyệt (đặc biệt Chrome) sẽ chặn việc đọc file/ảnh hoặc tải thư viện ngoài khi chạy theo cách này, khiến nút tải ảnh/file không hoạt động hoặc AI không phân tích được.
Cách khắc phục: chạy trang qua một server cục bộ thay vì double-click mở file, ví dụ:
  • Mở Terminal/CMD tại thư mục chứa file, gõ: python -m http.server 8000 rồi mở http://localhost:8000/index.html
  • Hoặc dùng tiện ích "Live Server" trong VS Code
  • Hoặc upload file lên một dịch vụ hosting tĩnh (Netlify, GitHub Pages...)
⚙️ Bước 1 – kết nối Google Sheets: dán URL Web App (Apps Script) bạn đã deploy vào ô bên dưới. Xem hướng dẫn lấy URL này trong file apps-script.gs đi kèm.
🤖 Bước 2 – kết nối AI miễn phí (Google Gemini): Lấy API key free tại aistudio.google.com/apikey (đăng nhập Google, bấm "Create API key"). Key chỉ lưu trong trình duyệt của bạn, không gửi đi đâu khác ngoài Google.

AI sẽ: đọc nội dung ảnh/văn bản thô → tách đúng câu hỏi, 4 đáp án, đáp án đúng → nhận diện Mã đề thi của từng câu → gộp các câu cùng mã đề lại với nhau và đánh số STT tăng dần trong mỗi mã đề. Gemini có hạn mức miễn phí hàng ngày, dùng vừa đủ để tránh bị chặn tạm thời.
Lưu ý: bạn cần tự nhập API key của riêng mình (mỗi người dùng key riêng để không bị chia sẻ hạn mức free với người khác).

Ảnh sẽ được lưu kèm dòng dữ liệu. Bạn cần tự đọc và gõ nội dung câu hỏi/đáp án vào bảng bên dưới (trình duyệt không tự OCR được tiếng Việt chính xác). Mỗi ảnh tạo 1 dòng mới để bạn điền nội dung.

Link được thêm làm ghi chú nguồn; nội dung câu hỏi vẫn cần điền tay vào bảng.

- PDF/Word/TXT: nội dung văn bản sẽ được trích xuất thô vào 1 ô để bạn tách câu hỏi thủ công.
- Excel/CSV: nếu đã có cột giống mẫu (Mã đề, STT, Nội dung câu hỏi, Đáp án A-D, Đáp án đúng) sẽ tự map vào bảng.
