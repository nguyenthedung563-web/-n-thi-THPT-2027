2 Kết nối Google Sheets
Chưa có Client ID? Xem hướng dẫn tạo (5 phút)
Lấy từ URL: docs.google.com/spreadsheets/d/ID-Ở-ĐÂY/edit
? Tạo OAuth Client ID — từng bước
  1. Vào console.cloud.google.com → tạo Project mới (đặt tên gì cũng được).
  2. Vào menu APIs & Services → Library, tìm Google Sheets API → bấm Enable.
  3. Vào APIs & Services → OAuth consent screen → chọn External → điền tên ứng dụng + email → Save.
  4. Ở mục Test users, thêm chính email Gmail bạn sẽ dùng để đăng nhập.
  5. Vào APIs & Services → Credentials → Create Credentials → OAuth client ID.
  6. Chọn loại Web application.
  7. Ở mục Authorized JavaScript origins, thêm đúng địa chỉ trang web bạn đang mở (xem ở thanh địa chỉ trình duyệt, ví dụ https://claude.site hoặc http://localhost:xxxx).
  8. Bấm Create → copy chuỗi Client ID (dạng ...apps.googleusercontent.com) → dán vào ô bên trên.
Lưu ý: vì app ở chế độ Test, chỉ những email bạn thêm ở bước 4 mới đăng nhập được. Muốn cho người khác dùng, thêm email họ vào Test users tương tự.
